# SuMoney Frontend

React + TypeScript frontend for the SuMoney backend.

## Start

```bash
npm install
cp .env.example .env
npm run dev
```

Frontend: http://localhost:5173  
Backend API: http://localhost:8888

## Structure

```text
src/
  app/
  components/
  constants/
  hooks/
  pages/
  services/
  store/
```
